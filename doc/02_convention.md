# 컨벤션 (Convention)

## 네이밍 (Naming)

* PascalCase
* camelCase
* snake_case
* kebab-case
* MACRO_CASE

## 가이드라인 (Guideline)

* 클래스 (Class)
* 함수 (Function)
* 변수 (Variable)
* 프로퍼티 (Property)
* 프라이빗 (Private)
* 소스 파일 (SourceFile)
* 애셋 파일 (AssetFile)
* 디렉토리 (Directory)

## 약자 (Abbreviation)

* Identifier -> Id
* Information -> Info

## 두문자어 (Acronym)

* UserInterface -> UI
* FileTransferProtocol -> FTP

## 특수 문자 (SpecialCharacters)

perl

* 코멘트 `#`
* 스트링 `$`
* 리스트 `@`
* 오브젝트 `%`
